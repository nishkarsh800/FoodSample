# FoodSample 😋

FoodSample is an app that gives you full information about the selected meal including its recipe🍦.

## TECH USED ⚙
 - Kotlin
 - RoomDB
 - Android Architecture Components 
 - MVVM
 - Coroutines
 - Dependency Injection
 - Retrofit
 - Recycler View
 - Glide
